# 🤖 AI Trading Assistant

An intelligent trading assistant that analyzes chart screenshots and provides technical analysis, including buy/sell recommendations, support/resistance levels, and risk assessment.

## 🚀 Features

- 📊 Upload any trading chart screenshot
- 🔍 AI-powered technical analysis
- 📈 Identifies chart patterns (Head & Shoulders, Double Top/Bottom, Flags)
- 🎯 Provides specific entry/exit levels
- ⚠️ Risk assessment and confidence scores
- 💡 Actionable trading recommendations
